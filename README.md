# Arcade de Minijogos - Disciplina de Algoritmos

Este projeto é um sistema de jogos desenvolvido em **Linguagem C** como requisito para a disciplina de Algoritmos. O objetivo é demonstrar o uso prático de estruturas de controle de fluxo e lógica de programação.

##Emanuelly Soares Magalhães

##Jogos Inclusos

### 1.Pergunta e Resposta (Capitais)
Um quiz de 5 perguntas sobre geografia. 
* **Lógica utilizada:** Laços `for` para as rodadas e `if/else` para validação de acertos.

### 2.Cobra na Caixa!
Jogo de sorte e estratégia em turnos onde dois exploradores buscam o botão de saída em uma tumba.
* **Lógica utilizada:** Aleatoriedade com `rand()`, controle de caixas abertas com flags e alternância de turnos.

### 3.Gousmas War
Batalha estratégica focada em gestão de fúria. O objetivo é levar a fúria do inimigo ao limite sem destruir suas próprias criaturas.
* **Lógica utilizada:** Gestão de estados (fúria), condições complexas de vitória e ações de redistribuição (dividir).

##Conceitos Aplicados
* **Estruturas de Repetição:** `while`, `do-while` e `for`.
* **Estruturas Condicionais:** `if`, `else if` e `else`.
* **Validação de Dados:** Tratamento de entradas inválidas para evitar travamentos.
